Instructions for installing interactives

  • Unzip/Extract the file
  • Open the file folder and click on `index.html` and open it using any browser. We recommend Firefox
    • Note : Due to Security Updates by latest browsers some of the tools won't run as expected. In order to ensure they run properly
  • Open Firefox Browser
  • Type about:config in address bar
  • Click on Accept The Risk and Continue
  • Click Show all
  • Search for security.fileuri.strict_origin_policy
  • Turn the value which is True to False
